# 渔貂宋体
一款符合中国大陆汉字字形风格，且明确版权公有、可供免费商用的宋体。名称来源：打鱼→fisher→渔貂。

本字体依照开源字体许可SIL Open Font License授权协议（详见库内“SIL Open Font License v1.1.txt”）发布于GitHub。
